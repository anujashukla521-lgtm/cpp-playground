## Object-Oriented Programming (OOP) in C++

This folder contains C++ programs that demonstrate the core concepts of Object-Oriented Programming.

## Concepts Covered
- Class and Object
- Constructors (default and parameterized)
- Destructor
- Encapsulation
- Access Specifiers (public, private, protected)
- Pointer to Object
- this Pointer

### Purpose
These programs are written for learning and practice purposes to build a strong foundation in C++ OOP concepts, which are commonly asked in technical interviews.

## Inheritance in C++

This folder contains C++ programs that demonstrate the concept of inheritance, which allows one class to acquire the properties and behavior of another class.

## Concepts Covered
- Single Inheritance
- Multilevel Inheritance
- Multiple Inheritance
- Constructor Calling Order
- Use of Base and Derived Classes

### Purpose
These programs are written to understand code reusability and the relationship between base and derived classes in C++. Inheritance is a fundamental concept of Object-Oriented Programming and is commonly tested in technical interviews.

Each program focuses on one type of inheritance for clarity and better learning.
